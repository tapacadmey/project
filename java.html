    3.Exception Handling:

    Exception is an abnormal condition.Exception is mainly two types Runtime and Exception.The Exception Handling in Java
    is one of the powerful mechanism to handle the runtime errors so that the normal flow of the application can be maintained.
    The core advantage of exception handling is to maintain the normal flow of the application. An exception normally
    disrupts the normal flow of the application; that is why we need to handle exceptions.
